Bubble Sort

Use this web page to study the Bubble Sort algorithm.

By comparing all items in a list to each other, the items can arranged in ascending order after sufficient comparisons between items are made.

Of course, this is not the most efficient sorting algorithm! At best, the efficiency for this algoirhtm can be described as O(n). Average and worst cases for this algorithm are the same: 0(n*n).

An explanation of the above "Big O" notations:

  • O(n): As the number of items in the list grow, the execution time of the Bubble Sort algorithm grows linearly in direct proportion to n (the size of the array).
  • O(n*n): As the number of items in the list grow, the execution time of the Bubble Sort algorithm grows quadratically in direct proportion to n (the size of the array).
